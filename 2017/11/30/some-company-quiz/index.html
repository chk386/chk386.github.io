어떤 회사 입사 퀴즈

2017-11-30
anipang, quiz

페북에서 우연히 채용 퀴즈를 보고 풀어봄
이런 문제들을 잘 안풀어봐서 난이도는 잘 모르겠다.
테스트 케이스 작성까지 총 2시간(조금 넘게?) 걸렸다.
BFS로 풀었으면 더 빨리 풀었을라나?

문제

입력

2 4 1 2 1
3 4 2 3 3
2 4 1 2 2
4 4 4 1 2
4 2 3 3 2

출력

0 0 0 0 0
2 0 0 0 0
3 0 0 0 0
2 0 0 2 0
4 0 1 3 0

조건

  1. 타일판은 5 × 5
  2. 타일 종류는 1 ~ 4의 네 가지
  3. 가로나 세로로 3개 이상 같은 타일이 연속될 경우 사라지고, 그 자리에는 위쪽의 타일들이 내려와서 메꿉니다.
  4. 내려오면서 비게 된 자리는 0으로 채워집니다.
    그냥 애니팡이라고 생각하면됨

풀이

  1. 주어진 2차원 배열(입력값)을 순차적으로 탐색
  2. 빈 타일 위쪽에 비어있지 않은 타일이 있으면 위에 타일을 아래로 내린다.
  3. 2차원 배열을 벗어나지 않고 위아래, 왼쪽오른쪽을 찾아서 3개 모양이 같은 걸 찾고 3개 값을 0으로 바꾼다.
  4. 1~3을 루프 돌린다. 2,3에 만족하는 타일이 없을경우 종료./
